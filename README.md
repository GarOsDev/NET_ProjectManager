# NET_ProjectManager

Project developed from scratch, focused on designing and programming a web application for client management. 
Based on the category of the registered/logged-in user, a set of specific functionalities will be unlocked, 
allowing them to perform certain tasks within the application.

Both the design and the development process were fully procedural, adapting progressively to the evolving 
requirements until shaping a functional application with clearly defined objectives.
